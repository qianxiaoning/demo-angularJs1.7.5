### simple_project 精简项目
```
src/
    components/ 组件
    config/
        dict.js 一个全局变量的run方法
        router.js 路由表
        validation.js 'angular-validation'的config配置
    controllers/ 控制器
    data/ 数据mock文件
    directives/ 指令
    filters/ 过滤器
    image/ 图片
    pages/ 页面
    services/ 服务（公共方法）
    app.js 入口函数
    common.less 公共样式文件
    index.html html模板文件
```
